This MB-KODI-ADDONS repository (https://m-borsch.github.io/MB-KODI-ADDONS/) is published as a "wrapper" repo that you do not directly install into Kodi. Instead it serves up a list of publicly available Kodi repositories as well as containing a zip with sample files to help customize your Kodi install.

Installation Instructions
1 - In Kodi use the "File Manager" to add a new 'source' pointing to this file: https://m-borsch.github.io/MB-KODI-ADDONS
2 - In Kodi use the "Addons" - "Install from zip" option to install the repositories listed below utilizing the 'source' that you added in the previous step
3 - In Kodi use the "Addons" - "Install from repo" option to install the assocated 'addon' from each respective repository that you added in the previous step
4 - (Optionally) In Kodi use the "Addons" - "Install from zip" option to install the Video Addons listed below
5 - In Kodi you can use the "File Manager" to copy the MB-KODI-SUPPORT-FILES.zip file to your local install of Kodi
6 - In Kodi, in the "Display" section you can use the sample background file from MB-KODI-SUPPORT-FILES as a custom Kodi background
7 - In Kodi you can use the "File Manager" to overwrite the favourites.xml file in the profile directory by copying the file from MB-KODI-SUPPORT-FILES as a start for your Kodi favourites. Note: requires a Kodi restart to load.
